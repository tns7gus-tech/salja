# SALJA 글로벌 웹페이지 총괄 기획서 (MVP)

## 1. 프로젝트 개요
- 프로젝트명: **SALJA**
- 티커명: **SALJA**
- 핵심 메시지: "죽지 말고, 살자. 오늘 하루만 같이 버티자."
- 목적: 우울·무기력·비교 피로를 겪는 청춘에게 짧지만 즉시 체감 가능한 정서적 위로를 제공하는 글로벌 에이전트 웹페이지 구축

## 2. 배경과 문제정의
- 대한민국은 높은 자살률과 청년 정신건강 위기라는 사회적 배경이 존재함.
- 많은 사용자가 SNS 비교, 경제적 압박, 관계 단절, 무기력으로 일상을 버티기 어려워함.
- 긴 상담 이전에, "지금 이 순간 버틸 수 있게 하는 짧은 문장과 행동"이 필요함.

## 3. 서비스 미션
- 미션: 힘든 사람에게 "오늘 하루를 버틸 이유"를 건네는 디지털 정서 안전지대 제공
- 톤앤매너: 따뜻함 + 솔직함 + 밈 감성 + 과장 없는 위로
- 차별점: 유머(밈)와 위로를 결합해 진입장벽을 낮추고, 즉시 실행 가능한 작은 행동까지 연결

## 4. 핵심 가치 제안 (Value Proposition)
- 30초 안에 위로를 받는다.
- 무거운 주제를 무겁기만 하지 않게 전달한다.
- "말"에서 끝나지 않고 "오늘의 작은 행동 1개"로 연결한다.
- 다국어 지원으로 글로벌 사용자 확장 가능

## 5. 타깃 사용자
- 1차: 18~34세 청년층 (한국)
- 2차: 글로벌 MZ 세대 (정서 피로, SNS 피로, 무기력 경험자)
- 주요 상황
  - SNS 비교로 자존감이 흔들릴 때
  - 중독적 습관(무한 스크롤, 도파민 루프)에서 벗어나고 싶을 때
  - "오늘만 버티고 싶다"는 감정이 강할 때

## 6. 핵심 콘텐츠 포맷 (고정 템플릿)
SALJA 1회 출력 = 아래 5개 묶음
1. 웃긴 캡션 (밈 스타일)
2. 위로 문장 (짧고 직설적)
3. 짧은 시 (2~4행)
4. 고전/명언 의역 (부담 없는 현대어)
5. 오늘의 작은 행동 1개 (즉시 실행)

### 예시 포맷
- Meme Caption: "인생 난이도: 하드모드인데 튜토리얼 없음"
- Comfort Line: "지금 버티는 것만으로도 이미 잘하고 있어."
- Short Poem: "오늘의 나는 / 완벽하진 않아도 / 포기하지는 않는다"
- Reframed Quote: "폭풍은 영원하지 않다. 지금의 밤도 지나간다."
- One Small Action: "물 한 컵 마시고, 어깨를 10초 펴기"

## 7. 페이지 구조 (MVP 랜딩 1페이지)
1. Hero Section
- **SALJA** 티커를 가장 크게 강조
- 서브카피: "Stay Alive, Laugh A Little, Keep Going."
- CTA: `오늘의 SALJA 받기`

2. Live Card Section
- 5개 콘텐츠 묶음을 카드 형태로 출력
- 새로고침 버튼: `다시 받기`
- 저장 버튼: `문장 저장`

3. Why SALJA Section
- 기획 배경(청년 위기, 비교 피로, 무기력)
- SALJA가 제공하는 도움 3가지 요약

4. Global Language Section
- 언어 선택 버튼/드롭다운 제공
- 기본 언어: 한국어(ko)
- 우선 지원: 영어(en), 일본어(ja), 스페인어(es)

5. Safety Section
- 위기 시 도움 링크/문구 노출
- 국가별 긴급 지원 번호 또는 연결 가이드 (추후 확장)

## 8. UI/UX 방향

### 8-1. 티커 **SALJA** 강조 UI
- 헤더 좌측 고정 로고: `SALJA`
- Hero 중앙 대형 타이포: `SALJA` (화면 내 시각 우선순위 1위)
- 배지 시스템: `Ticker: SALJA`
- 카드 워터마크: 배경에 반투명 `SALJA`

### 8-2. 분위기와 스타일
- 키워드: "다정한 네온", "도시의 밤", "무겁지만 따뜻함"
- 컬러 제안
  - Primary: #00E5A8
  - Deep BG: #0F172A
  - Accent: #FFD166
  - Alert Soft: #FF6B6B
- 폰트 방향
  - 제목: 강한 존재감의 산세리프
  - 본문: 가독성 높은 글로벌 폰트

### 8-3. 다국어 버튼 UX
- 헤더 우측: `Language` 버튼
- 클릭 시 언어 목록 표시 (ko/en/ja/es)
- 언어 변경 즉시 UI 텍스트 및 콘텐츠 템플릿 반영
- 브라우저 언어 자동 감지 + 수동 변경 우선

## 9. 기능 정의 (MVP 기준)
- 오늘의 SALJA 생성/노출
- 새 콘텐츠 다시 생성
- 다국어 선택
- 카드 저장(텍스트 복사 또는 이미지 저장 중 택1)
- 공유 버튼 (X, Instagram 스토리 문구 복사용)

## 10. 콘텐츠 운영 가이드
- 금지: 죄책감 유발 문장, 단정적 훈계, 위험행동 미화
- 권장: 짧고 명확한 위로, 생활밀착 행동 제시, 문화권 중립 표현
- 길이 기준
  - 밈 캡션: 1문장
  - 위로 문장: 1~2문장
  - 짧은 시: 2~4행
  - 명언 의역: 1문장
  - 작은 행동: 1개

## 11. 안전/윤리 가이드
- 본 서비스는 치료/진단 대체가 아님을 명시
- 고위험 신호(극단적 표현) 감지 시 안전 안내 우선 노출
- 국가별 도움 리소스 연결 정책 수립
- 커뮤니티/댓글 기능 도입 시 모더레이션 필수

## 12. KPI (초기)
- 방문자 대비 `오늘의 SALJA 받기` 클릭률
- 콘텐츠 재생성(다시 받기) 횟수
- 공유/저장 전환율
- 언어별 리텐션
- 이탈 구간 (Hero 이후, Live Card 이후)

## 13. 빠른 제작 일정 (초단기)
- Day 1: 카피 확정 + 와이어프레임 + 비주얼 톤 결정

- Day 3: QA + 모바일 최적화 + 배포
- Day 4+: 콘텐츠 품질 개선 + 언어 확대 + 공유 기능 고도화
